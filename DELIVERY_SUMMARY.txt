## 🚀 Execution Steps

### Step 1: Development Machine

```bash
# 1. Setup
cd currency-detection
bash scripts/setup-dev.sh
source venv/bin/activate

# 2. Prepare data (2 min)
python scripts/prepare_dataset.py

# 3. Train model (15-30 min)
python scripts/train_model.py

# 4. Convert to TFLite (2-3 min)
python scripts/convert_tflite.py

# 5. Test (1 min)
python scripts/test_inference.py
```

**Output:** `models/currency_model.tflite` (2.5-3.5 MB)

### Step 2: Raspberry Pi

```bash
# 1. Setup
ssh pi@raspberry
bash scripts/setup-rpi.sh

# 2. Copy model
scp models/currency_model.tflite pi@raspberry:~/currency-detection/models/

# 3. Run detection
python src/inference.py
```

**Output:** Real-time JSON detections
